## Getting Started

First, run the Decap server:

```bash
npx decap-server

# output:
#
# info: Decap CMS File System Proxy Server configured with [path/to/this/repo]
# info: Decap CMS Proxy Server listening on port 8081
```

Second, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

- Open [http://localhost:3000](http://localhost:3000) with your browser to read your posts.

- Open [http://localhost:3000/admin](http://localhost:3000/admin) with your browser to manage your posts and images.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
